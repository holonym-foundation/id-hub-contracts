Verifiers generated by Zokrates have a few changes that must be done to be compatible with this:
     *import ./PairingAndProof.sol instead of using the in-file Pairing and Proof struct Zokrates generates (otherwise there will 
      be conflicts between Pairing structs in different files)
     *verifyEncoded function that converts bytes to struct or otherwise serialized inputs. It must implement IVerifier.sol. One example is:

# AssertLeafContainsCredsVerifier.sol
function verifyEncoded(Proof calldata proof, uint[] calldata input_) public view returns (bool r) {
        // (Proof memory proof, uint[25] memory input) = abi.decode(b, (Proof, uint[25]));
        uint[25] memory input;
        for (uint i = 0; i < 25; i++) {
            input[i] = input_[i];
        }
        return verifyTx(proof, input);
    }
